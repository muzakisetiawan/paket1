Tutorial Membuat CBT Sekolah

Website CBT menggunakan GitHub

STEP 1 — Buat Repository GitHub

1. Buka GitHub

Masuk ke:

Login akun kamu

2. Klik tombol:

New repository

3. Isi seperti ini:

  • Nama repository: cbt-sekolah
  • Pilih: Public
  • Centang: Add README

Klik:

Create repository

STEP 2 — Upload File Website Pertama

Klik:

Add file → Create new file

Nama file:

index.html

Isi:

<!DOCTYPE html> <html> <head> <title>CBT Sekolah</title> </head> <body> <h1>CBT Online Sekolah</h1> <p>Website latihan ulangan</p> <a href="login.html">Login</a> <br><br> <a href="register.html">Register</a> </body> </html>

Klik:

Commit changes
